# AutoSalon WinForms

Клиентское приложение Windows Forms для курсовой работы «Информационная система автосалона».

## Что реализовано

- авторизация по e-mail и паролю из `App.config`;
- роли: руководитель, менеджер, склад, сервис, администратор;
- главное окно с новой темной боковой панелью, карточками статистики и плитками модулей;
- разделы: клиенты, автомобили, сделки, тест-драйвы, услуги, сотрудники, отчеты;
- обработчики событий поиска, добавления, изменения, удаления, сохранения, отмены, проверки подключения, экспорта отчетов и выхода;
- проверки данных: VIN, ИНН, телефон, e-mail, даты, длительность тест-драйва, цены;
- экспорт отчетов в CSV, TXT, TSV, JSON, XML.

## Подключение к базе

В `App.config` используется строка:

```xml
Data Source=.\SQLEXPRESS;Initial Catalog=AutoSalonDB;Integrated Security=True;TrustServerCertificate=True
```

При необходимости замените имя сервера SQL Server на свое.
